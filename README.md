# growth-agent

The **growth layer** for Kugiro. Kugiro tells a candidate *how well they match
jobs today*; this agent tells them *what to learn to match better tomorrow* —
as a ranked, costed, auditable roadmap.

It is an **MCP client of Kugiro**: it pulls the candidate's real scanned jobs
over MCP, figures out which in-demand skills they're missing, and scores the
payoff of closing each gap. The scoring is **generated by Claude and executed
inside an isolated Daytona sandbox** — so the numbers come from real, auditable
code, not a black-box guess.

## Why Daytona (the load-bearing part)

The gap-and-uplift scoring is *generated code*. We never run that generated
code on our own machine — it runs in a fresh, isolated Daytona sandbox that is
destroyed afterward. Per candidate, the analysis lives and dies in its own box.
That's a real isolation story, not decoration: you can show the exact program
that ran and the numbers it produced.

## Architecture

```
Kugiro (Next.js + MCP server)  ──MCP──>  growth-agent (this repo)
  get_search_profile  → candidate profile     1. read profile + real jobs
  list_jobs / get_job → real scanned jobs      2. aggregate market demand
                                               3. Claude GENERATES scoring code
                                               4. run it in a Daytona sandbox
                                               5. return ranked learning roadmap
```

## Run it

Requires Python 3.13 (3.14 lacks some wheels — see SETUP.md).

```bash
# 1. install
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt

# 2. configure keys (see SETUP.md)
copy .env.example .env   # then fill in real keys

# 3a. OFFLINE demo — works anywhere, no Kugiro needed:
.\.venv\Scripts\python.exe run_agent.py

# 3b. LIVE demo — pulls real data from a running Kugiro:
.\.venv\Scripts\python.exe run_agent.py --kugiro
```

The offline demo is the **guaranteed** path: it exercises Claude code-gen + the
Daytona sandbox end-to-end without depending on the dashboard being up.

## What you get

A ranked roadmap: for each missing in-demand skill — how much it lifts fit, what
% of the candidate's target jobs ask for it, and a concrete course with real
cost and duration. Highest-ROI move first.
